# FF Logs 战斗记录分析器 (Web 版)

这是一个纯前端的 Web 应用程序，用于从 FF Logs 导出 BOSS 技能与伤害时间轴。

## 使用说明

点击网址https://yilegendoflink.github.io/XIVtimelineMaker/ 进入使用。需要先登录fflogs页面进行授权。

v2 api有免费用量限制，如果用完了需要自行提供clientID，网站内有教程

## 关于技能类型（物理/魔法）和翻译问题

已解决，可喜可贺

伤害类型理论上有一点点概率出问题，出问题了再说

## 关于时间轴一体机导入模式

开启一体机导入模式后生成的表格文件将在格式上更加便于复制粘贴时间轴进入無川@幻影群岛与屿川@幻影群岛大佬制作的减伤一体机表格中，非常建议在复制前自行手动对自动记录的伤害事件进行简单筛选以适配复杂多变的实际副本情况。

减伤一体机发布网址：https://ngabbs.com/read.php?tid=45896202

## 注意事项
- 首次使用需要点击“登录 FF Logs”进行授权。
- 授权有效期通常较短，如果遇到 401 错误，请点击“断开连接”重新登录。
