如何获取 FF Logs Client ID

  1. 登录 FF Logs 网站。
  2. 点击右上角头像，选择 Settings (设置)。
  3. 在左侧菜单底部找到 Web API 并选择 Manage your v2 clients (或者直接访问 API Clients)。
  4. 点击 Create Client (创建客户端)。
  5. 填写信息：
    • Name: 任意填写 (例如: Timeline Exporter)
    • Redirect URL:
      务必填写的完全一致
    • Public Client: 勾选 ✅ (非常重要！)
  6. 点击创建后，复制显示的 Client ID。
  7. 将 Client ID 粘贴到本工具的输入框中即可。
